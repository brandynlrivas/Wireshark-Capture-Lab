This project captures and analyzes live network traffic using Wireshark on macOS. The objective of this lab was to capture key network protocols and demonstrate effective packet-level analysis using Wireshark display filters. The lab focuses on understanding how common protocols operate in real-world network environments, particularly in modern networks that primarily enforce HTTPS encryption.
